# claude-api tutorial

Demo project companion for the article [Claude Code /claude-api (You Can Now Build Claude Apps 3x Faster)](https://medium.com/@joe.njenga/claude-code-claude-api-you-can-now-build-claude-apps-3x-faster).

This folder will contain example snippets and notes demonstrating how to use the `/claude-api` skill in Claude Code to build apps powered by the Anthropic SDK.
