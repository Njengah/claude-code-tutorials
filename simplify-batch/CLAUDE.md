# Project: simplify-batch-demo

## Code Standards
- Use ES modules (import/export) not CommonJS (require/module.exports)
- Use const/let, never var
- No console.log in production code
- Always validate inputs on API endpoints
- No raw SQL string concatenation — use parameterized queries
- Remove all commented-out code and TODO stubs

## Patterns
- Use array methods (map, filter, reduce) over for loops
- Use async/await over callbacks
