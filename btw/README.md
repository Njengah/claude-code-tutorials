# btw tutorial

Demo project companion for the article [I Tried (New) Claude Code /btw Command (Now You Can Chat As You Code)](https://medium.com/@joe.njenga/i-tried-new-claude-code-btw-command-now-you-can-chat-as-you-code).

This folder contains notes and examples showing how to use the `/btw` command in Claude Code to ask quick side questions without interrupting the main task.
