# loop tutorial

Demo project companion for the article [Claude Code /loop — How I Create New Native Autonomous Loops That Work!](https://medium.com/@joe.njenga/claude-code-loop-how-i-create-new-native-autonomous-loops-that-work).

This folder contains a minimal Node.js project you can use to test Claude Code’s `/loop` command, which runs autonomous background loops that monitor builds, deployments, PRs, and more.
